1. Metode KNN

Klasifikasi data KNN adalah salah satu aplikasi dari metode K-Nearest Neighbors (KNN) dalam pembelajaran mesin. Klasifikasi data KNN mengambil sekelompok data dengan fitur yang mirip dan mengklasifikasikannya ke dalam kelas atau label yang sama.

Dalam KNN, setiap data pada dasarnya diwakili sebagai sebuah titik dalam ruang fitur multidimensi. Ketika KNN melakukan klasifikasi data baru, ia akan mencari sekelompok tetangga terdekat dari data baru tersebut dalam ruang fitur tersebut. KNN kemudian mengambil mayoritas label dari tetangga terdekat sebagai label yang akan diberikan pada data baru tersebut. Oleh karena itu, KNN disebut sebagai metode “lazy learning” karena tidak memerlukan proses pelatihan yang kompleks seperti halnya dengan metode pembelajaran mesin yang lainnya.

Langkah 5

  1. Langkah pertama adalah menginisialisasi objek knn dari kelas KNeighborsRegressor, yang digunakan untuk melakukan regresi dengan algoritma K-Nearest Neighbors. Selanjutnya, kita melakukan randomisasi indeks dari dataset dengan menggunakan np.random.permutation() dan menetapkan seed 1 agar hasilnya konsisten.

  2. Kemudian, dataset dibagi menjadi dua bagian: 75% untuk training dan 25% untuk testing. Data training disimpan dalam variabel train_index dan data testing disimpan dalam variabel testing_data.

  3. Setelah itu, melakukan training model menggunakan data training dengan memanggil method fit() pada objek knn. Argumen input berupa fitur-fitur yang digunakan (yaitu age, sex, cp, trestbps, chol, fbs, restecg, thalach, exang, oldpeak, slope, ca, dan thal) dan labelnya adalah target.

  4. Selanjutnya, model melakukan prediksi pada data testing dengan memanggil method predict() pada objek knn. Argumen input berupa fitur-fitur dari data testing yang ingin diprediksi.

  5. Selanjutnya, akurasi model dihitung dengan mengukur berapa persen data testing yang berhasil diprediksi dengan threshold default 0.5.

  6. Terakhir, hasil akurasi model akan dicetak.

knn = KNeighborsRegressor()

# before we split, make sure you randomize the dataset
np.random.seed(1)
index_shuffled = np.random.permutation(df_copy.index)
shuffled_car_data = df_copy.loc[index_shuffled]

# We will use 75% of the data for training and the remaining 25% for testing
train_index = int(shuffled_car_data.shape[0] * 0.75)
training_data = shuffled_car_data.iloc[0:train_index]
testing_data = shuffled_car_data.iloc[train_index:]

# fit the model with the training data
knn.fit(training_data[['age', 'sex', 'cp', 'trestbps', 'chol', 'fbs', 'restecg', 'thalach', 'exang', 'oldpeak', 'slope', 'ca', 'thal']], training_data['target'])

# predict the target of the testing data
prediction = knn.predict(testing_data[['age', 'sex', 'cp', 'trestbps', 'chol', 'fbs', 'restecg', 'thalach', 'exang', 'oldpeak', 'slope', 'ca', 'thal']])

# calculate the accuracy of the model
accuracy = np.mean(np.abs(prediction - testing_data['target']) <= 0.5)

print(f"Akurasi model KNN: {accuracy:.2%}")

Akurasi model KNN: 92.11%
